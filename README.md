# Sankhini Personal Website

Single-file personal website with a quiet notebook aesthetic.

## File
- `index.html` - contains content, styles, and scripts.

## Edit Content
Open `index.html` and edit only the block labeled:
`EDITABLE CONTENT BLOCK ONLY`

You can update:
- name + tagline
- currently list
- life story eras
- writing links
- contact email/links
- poem text

## Features
- Day/night mode with saved preference
- Watercolor wash accents near hero
- Canvas watercolor splash on click anywhere
- Poem easter egg note in bottom-right corner

## Run Locally
```powershell
cd s:\python\SRC\personal_website
python -m http.server 5500
```
Open `http://localhost:5500`
